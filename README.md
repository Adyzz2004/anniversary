## How to Deploy to GitHub Pages üöÄ

1. **Create a GitHub Repository**
   - Go to GitHub and create a new repository
   - Name it something like `anniversary-website` or `1-year-anniversary`

2. **Upload Files**
   - Clone the repository to your computer
   - Copy all files (`index.html`, `styles.css`, `script.js`) to the repository folder
   - Commit and push:
     ```bash
     git add .
     git commit -m "Initial commit - Anniversary website"
     git push origin main
     ```

3. **Enable GitHub Pages**
   - Go to your repository on GitHub
   - Click on **Settings**
   - Scroll down to **Pages** section
   - Under **Source**, select `main` branch
   - Click **Save**
   - Your site will be live at: `https://yourusername.github.io/repository-name`
